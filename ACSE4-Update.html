Travis is your friend

  • Travis will catch pushed commits which don't run
  • Look for the red cross/green tick
  • Can make GitHub pull requests have to pass
  • Can make merges to master have to PR
  • Use mark.xfail & mark.skip

Syntax errors

  • accidental symbols break python code
  • tabs or spaces, not both
  • Travis can catch these

Relative imports

  • Python only looks in a limited number of places for imports
  • Usually current directory and system directories
  • For packages may need to use relative imports
  • Eg. in __init__.py we have
    • from .tool import *
